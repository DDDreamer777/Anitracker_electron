# ZAPIS-Pi Controller (PC端)

斑马鱼行为学实验 - 树莓派 LED/音频控制器

## 环境要求

- Node.js 18+
- Windows/macOS/Linux
- 与树莓派在同一局域网

## 安装

```bash
cd desktop
npm install
```

## 运行

```bash
npm start
```

## 界面说明

### 树莓派 IP 配置
- 输入树莓派的局域网 IP 地址（默认 192.168.2.118）

### LED 昼夜循环控制
- **昼时间**：白光持续的分钟数（默认 10 分钟）
- **夜时间**：灯光熄灭的分钟数（默认 10 分钟）
- **循环次数**：0或空=无限循环，其他数字=指定次数后自动停止
- 点击「开启循环」开始自动昼夜交替
- 点击「停止循环」终止循环

### LED 手动颜色控制
> ⚙️ 功能默认关闭，如需启用请参考下方说明

如需启用手动颜色控制，请编辑 `index.html`，取消注释 `<!-- LED Manual Control -->` 部分（第 47-58 行），保存后重启应用。

- 支持：红、绿、蓝、白、关闭
- 手动颜色命令会终止当前的昼夜循环

### 音频播放
- 输入 WAV 音频文件名（放在树莓派 `./audio_files` 目录）
- 点击「播放」发送播放指令

## TCP 端口

| 功能 | 端口 |
|------|------|
| LED 控制 | 9999 |
| 音频播放 | 9998 |

## 协议格式

### LED
```
cycle,day_min,night_min      → 开启昼夜循环（无限）
cycle,day_min,night_min,n    → 开启昼夜循环，执行n次后停止
stop                         → 停止循环
```

### 音频
```
文件名.wav               → 播放音频
```
